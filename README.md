# Cultural Documentary Archive

A modern, responsive, and SEO-optimized platform designed to host long-form cultural documentaries, rituals, and historical records.

## 🏛 Project Vision
This archive serves as a digital bridge between past and future, documenting disappearing traditions through high-fidelity storytelling and modern web design.

## 🚀 Technical Features
- **Modern Heritage Aesthetic:** A custom UI using a "Parchment & Charcoal" color palette.
- **AdSense Ready:** Pre-configured semantic slots for Google AdSense integration.
- **SEO Optimized:** Structured with semantic HTML5 tags (`<article>`, `<nav>`, `<header>`) and a valid `sitemap.xml`.
- **Mobile First:** Fully responsive grid layout for viewing on any device.
- **Fast Performance:** Lightweight Vanilla HTML, CSS, and JS for near-instant load times.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Variables)
- **Interactivity:** Vanilla JavaScript
- **SEO:** XML Sitemaps, Robots.txt, and Metadata optimization

## 📂 Folder Structure
- `index.html`: The homepage featuring the documentary grid.
- `article.html`: The template for long-form documentary stories.
- `about.html`: Mission statement and curator information.
- `privacy.html`: Legal privacy policy (Required for AdSense).
- `style.css`: Global design system.
- `sitemap.xml`: Index for Google Search Console.
